# Aircraft Performance Engineering Tool

A Python-based aerospace engineering tool for calculating fundamental aircraft performance parameters.

## Features
- Wing Loading calculation
- Thrust-to-Weight Ratio analysis
- Lift force estimation (aerodynamic model)
- Modular engineering structure

## Engineering Concepts Used
- Basic aerodynamics
- Newtonian mechanics
- Aircraft performance theory

## Project Structure
- main.py → User interface
- calculations.py → Engineering formulas
- constants.py → Physical constants

## Purpose
This project was developed as part of my preparation for a career in aircraft engine and aerospace engineering, with focus on practical application of Python in engineering analysis.
